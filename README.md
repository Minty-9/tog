Triumph Delicious Kitchen (TDK) — Business Website

A modern, responsive business website for Triumph Delicious Kitchen, a Nigerian foodstuff export and catering rental company serving customers in Canada, USA, UK, and France.

### 🛠️ Built With

  * **HTML5** — Semantic structure for SEO and accessibility
  * **CSS3** — Custom properties, Flexbox, Grid, and keyframe animations
  * **Vanilla JavaScript** — High-performance logic with zero dependencies
  * **Google Fonts** — Professional pairing of Playfair Display and DM Sans

### ✨ Features

  * **Global Responsiveness** — Seamless experience on mobile, tablet, and desktop
  * **Smart Navigation** — Smooth-scroll logic for improved user flow
  * **Dynamic Hero Section** — High-impact animated marquee ticker for brand visibility
  * **Dual-Service Showcase** — Specialized catalogs for foodstuff exports and catering rentals
  * **Interactive FAQ** — Accordion-style section to reduce customer support friction
  * **Conversion-Driven CTA** — Strategically placed contact and lead generation sections


### 🎯 Purpose

This project was developed for a client in the Nigerian food export industry. The primary goal was to establish a professional global presence that:

  * **Establishes Trust** with international customers in the diaspora.
  * **Streamlines Orders** by clearly categorizing export goods vs. local rentals.
  * **Optimizes Performance** through lightweight, framework-free engineering.

### 🚀 Getting Started

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/minty-9/tog.git
    ```
2.  Open the directory:
    ```bash
    cd tog
    ```
3.  Launch `index.html` in your browser.
